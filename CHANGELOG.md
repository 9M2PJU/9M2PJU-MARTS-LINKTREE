# Changelog

All notable changes to this project will be documented in this file.

## [v0.1.0] - 2026-01-20

### Added
- Initial release of MARTS Directory Linktree.
- Performance optimizations for mobile devices (reduced background blur, optimized animations).
- GPU acceleration hints for smoother scrolling and interactions.
- Explicit image dimensions to prevent Cumulative Layout Shift (CLS).
- "Back to Top" button.
- PWA installation prompt.
- Automated link syncing via GitHub Actions.
